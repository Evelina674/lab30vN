# Лабораторна робота №30

## Тема
Написання юніт-тестів з xUnit.

## Мета
Навчитися писати юніт-тести для власного коду за допомогою xUnit, використовуючи різні типи assertion та параметризовані тести.

## Варіант 7 - ShoppingCart

У роботі реалізовано клас `ShoppingCart`, який містить бізнес-логіку кошика покупок.

### Основні методи:
- `AddItem` - додає товар у кошик.
- `RemoveItem` - видаляє товар з кошика.
- `CalculateTotal` - обчислює загальну суму товарів.

## Структура проєкту

- `lab30vN` - основний проєкт з бізнес-логікою.
- `lab30vN.Tests` - тестовий проєкт з юніт-тестами.
- `lab30vN.sln` - solution-файл.

## Що перевіряють тести

У тестах перевірено:
- додавання товарів;
- підрахунок загальної суми;
- роботу з кількома товарами;
- видалення товару;
- спробу видалення неіснуючого товару;
- некоректні дані: порожня назва, від’ємна ціна, нульова кількість.

## Використані атрибути xUnit

- `[Fact]` — звичайний тест без параметрів.
- `[Theory]` — параметризований тест з різними вхідними даними.
